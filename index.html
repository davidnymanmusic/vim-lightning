̴ What is Vim?

Vim is a cloned version of Vi

Vi is a screen-oriented text editor originally created for the Unix operating system.

Vim stands for "Vi Improved", though orignally it stood for "Vi Imitation"

̴ Where is Vim?

Vi comes preinstalled on almost all Unix systems and most Linux

$ vi in the console to start

$ vimtutor will show you how to get started

̴ Why use Vim?

Can be used on all systems

Edit text directly in the terminal

Very efficient as you only use the keyboard

̴ Is Vim worth it?

Maybe not right now, but it's worth exploring and keeping in mind

Atom has a vim mode

It helps to think of it as a game with the keyboard as a controller

Who actually uses it?

  • OpenTable
  • AdRoll
